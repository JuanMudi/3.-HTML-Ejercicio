Requerimientos No Funcionales Principales

4.1 Aplicación Móvil (Online/Offline)

Capacidad de la aplicación para funcionar sin conexión, con sincronización de datos cuando se restablece la conectividad.

4.2 Aplicación Web

Desarrollo de una interfaz web para administradores con herramientas de visualización avanzada y generación de informes.
