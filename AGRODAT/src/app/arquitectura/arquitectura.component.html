Capas de la Arquitectura

5.1 Capa 1: Datos

Persistencia de datos y sincronización en MySQL 8.0.23 sin lógica de procedimientos almacenados.

5.2 Capa 2: Lógica de Negocio

Procesamiento de información con POJOs y patrón VO/DAO para la comunicación con la base de datos.

5.3 Capa 3: Conectividad Cliente Final

API utilizando Servlets y JSON en TOMCAT para la comunicación entre cliente y servidor.

5.4 Capa 4: Visualización

Diferentes vistas para usuarios finales y administradores, desarrolladas en Apache Cordova y React.
